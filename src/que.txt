React - Tag Input
Tag Input Component
Build a tag input where pressing Space adds the typed text as a tag. Each tag has an × button to remove it.
Requirements
 

Space adds a tag, × removes it
 
No duplicates — show an error message
 
Max 10 tags — disable input when reached
Contraints:
The UI must be responsive across different screen sizes.
Ensure the design looks clean, modern, and visually appealing (good spacing, clear tag styling).
